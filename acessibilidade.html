Diretrizes

As Diretrizes de Acessibilidade para Conteúdo Web (WCAG) 2.0 abrangem um vasto conjunto de recomendações que têm como objetivo tornar o conteúdo Web mais acessível. O cumprimento destas diretrizes fará com que o conteúdo se torne acessível a um maior número de pessoas com incapacidades, incluindo cegueira e baixa visão, surdez e baixa audição, dificuldades de aprendizagem, limitações cognitivas, limitações de movimentos, incapacidade de fala, fotossensibilidade bem como as que tenham uma combinação destas limitações. Seguir estas diretrizes fará também com que o conteúdo Web se torne mais usável aos utilizadores em geral.

    • Orientações básicas

    Algumas orientações básicas para criar uma página web acessível são:

    • Faça descrição alternativa das imagens, descrevendo de forma detalhada.
    • Não use apenas cores para destacar uma informação.
    • Simplifique seu texto, evite usar figuras de linguagem.
    • Crie áreas de clique maiores nos botões.
    • Separe os textos em blocos curtos.
    • Utilize o alinhamento à esquerda para os textos.
    • Coloque legendas nos vídeos.
    • Evite escrever o texto todo em maiúsculo, itálico ou negrito.
    • Não utilize fonte cursiva ou decorada.
    • Cuide com os contrastes das cores de texto e fundo.
    • Não utilize carrosséis de imagens automáticos.
    • Forneça mensagens de erro detalhadas nos formulários.
    • Não bloqueie a função zoom nas versões mobile.
